How It Works
  1. Create Session: Submit your content blocks, images, and design preferences
  2. Normalization: Content is validated and normalized automatically
  3. Submit for Layout: Trigger layout generation in the Gestalt Design Engine
  4. Download Artifacts: Retrieve generated layout specifications
API Information

The Content Intake Service provides REST APIs for programmatic access:

  • POST /v1/intake/sessions - Create intake session
  • GET /v1/intake/sessions/{id} - Retrieve session
  • POST /v1/intake/sessions/{id}/submit - Submit for layout
  • GET /v1/intake/sessions/{id}/artifacts - List artifacts
